Elementos e Atributos HTML

Seções

A tag <section> é usada para agrupar conteúdo relacionado. Ajuda a estruturar a página em partes lógicas e pode conter títulos, parágrafos, imagens e outros elementos.

Headings e Parágrafos

As tags <h1> a <h6> são usadas para criar títulos e os parágrafos são criados com a tag <p>. Os títulos indicam a hierarquia de informações na página, sendo <h1> o mais importante e <h6> o menos importante.

Título 1

Este é um parágrafo de texto.

Título 2

Mais texto do parágrafo.

Listas

As listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>) e são usadas para agrupar itens relacionados. Os itens são criados com a tag <li>.

  • Item 1
  • Item 2
  • Item 3
  1. Item A
  2. Item B
  3. Item C

Tabelas

As tabelas são usadas para exibir dados em uma grade. Elas consistem em linhas (<tr>), células de cabeçalho (<th>) e células de dados (<td>).

Nome Idade
João 30
Maria 28

Imagens

A tag <img> é usada para exibir imagens na página. O atributo "src" define o caminho da imagem e o atributo "alt" fornece um texto alternativo para acessibilidade.

Iframes

A tag <iframe> é usada para incorporar conteúdo externo, como vídeos do YouTube. Os atributos "width" e "height" definem as dimensões do iframe, e o atributo "src" define a URL do conteúdo incorporado.
